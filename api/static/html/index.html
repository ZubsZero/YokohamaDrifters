User Routes

Request Route URL Description
GET /users Fetches all users from DB and displays
POST /users Inserts new users to the DB
POST /users/login Upon examining the database to confirm the user's email and Pwdword, a JWT is generated for authenticated users
GET /users/:id Fetches specific user from DB by their userID & displays
PATCH /users/:id Updates specific user information
DELETE /users/:id Deletes specific user by the User ID.

Product Routes

Request Route URL Description
GET /products Fetches all products from DB and displays
POST /products Inserts new product to the DB
GET /products/:id Fetches specific product from DB by their Product ID & displays
PATCH /products/:id Updates/Changes specific product information
DELETE /products/:id Deletes specific product by the product ID.
